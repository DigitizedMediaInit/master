Digitized Media Init

Master Branch

Welcome to the Digitized Media Initiative master branch. Please feel free to download anything here, as it is free. Please follow the format below for use of the framework.

Framework Format

The following format is required for every webpage that you have the framework on.

<!DOCTYPE html>
<head>
    <meta charset="UTF-8">
    <title>Hey, this is my website title</title>

    <!--Styles-->
    <link rel="stylesheet" type="text/css" href="css/master.css">

    <!--Scripts-->
    <script type="text/javascript" src="js/jquery.min.js"></script>
    <script type="text/javascript" src="js/equalize.min.js"></script>
    <script type="text/javascript" src="js/jquery.scrollUp.min.js></script>
</head>
<body>
    <h1>My site's header</h1>
    <p>Some paragraph stuff to go after it (wooooooo)</p>
</body>
</html>
